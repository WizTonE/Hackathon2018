# 用戶端設計
## 使用目的
對於行李很多或行動不便的旅客，藉由使用我們的服務，可以從機場經過捷運到旅館房間都不用爬樓梯
## 大概操作流程
機場 ==> 捷運 ==> 旅館 ==> 完成

## 使用者主要的操作
1. 開始畫面
1. 選擇使用者語言
1. 輸入地址
1. 從機場導航到該地址(Here Map)
1. 機場導引圖(Google Map)
1. 捷運導引圖
1. 捷運出口導引圖 (Here Map)
1. 完成畫面
